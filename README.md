# Task Management Dashboard

## Overview

A React.js Task Management Dashboard that fetches tasks from the DummyJSON API and allows users to search, filter, sort, paginate, add, edit, delete, and manage tasks with Dark Mode support.

---

## Features

- Fetch 150 tasks from DummyJSON API
- Search tasks
- Filter (All / Completed / Pending)
- Sort tasks
- Pagination
- Add Task
- Edit Task
- Delete Task
- Task Detail Modal
- Summary Cards
- Loading / Error / Empty States
- Responsive UI
- Dark Mode

---

## Tech Stack

- React.js
- JavaScript (ES6)
- Tailwind CSS
- Axios
